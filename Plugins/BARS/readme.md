# BARS Plugin for FASA

## Overview

BARS is an addon for Microsoft Flight Simulator which utilitises taxi lights to guide aircraft safely to the runway or to a designated point in the airport.

## Installation

Bars is already configurated in the sector file, however you require an API key to access the BARS network. Simply create an account on the website and copy the API Link into the dedicated box in Euroscope.

```yaml
plugins:
  - bars
